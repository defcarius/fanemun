 <h1> Manual Deploy</h1>
 
<h5> Clone the repo </h5>

```
$ git clone https://github.com/Parcoil/DW GAMES.net
```

<h5> Install </h5>

```
$ npm install
```

<h5> Start the server </h5>

```
$ npm start
```
